What is CSS?

  1. CSS stands for Cascading Style sheets.
  2. CSS describes how HTML elements are to be displed on screen, paper, or in other media.
  3. Css is not neccesary CSS saves a lot of work. it can control the layout of multiple web pages all at once
  4. stylesheets are stored in CSS files

CSS can be added to HTML documents in 3 ways:

  • Inline - by Using the style attribute inside HTML elements1
  • Internal - by using a link element to link to an external CSS files.
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS, is to keep the tyles in extrnal CSS files. however, in this tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier for you to try it yourself.

When comparing CSS class vs ID, the diffrence is that CSS class applies a style to multiple elements. ID on th other hand, applies a style to one unique element. ID is also special in that you can use aa special URL to link directly to an element and it's used by javaScript.

In CSS, selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and ID. both are used to target elements to which a style should be applied.
